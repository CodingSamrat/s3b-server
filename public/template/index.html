Welcome to s3b-server

A Self Hosted S3 Bucket

Getting Started

1. Create admin user

$ s3b-server create-admin

2. Login as admin

$ s3b-server login

3. Create Bucket to get secret keys

{
    apiKey:******************,
    apiSecret:**************************,
    bucketId:******************
}

4. Install s3b for client

$ npm install s3b

5. Use secret keys to configure s3b
